> 东风第一枝 樱桃
>
> 清末近现代初·赵熙
>
> 玉子圆匀，珠胎点滴，几天红到如许。
> 
> 醉时三月登盘，霁色万星照树。
> 
> 筠笼上市，恰才听、卖花声住。
> 
> 算几村、杏子枇杷，一色淡青烟雨。
> 
> 曾梦入、上兰旧句。
> 
> 曾擎出、大明归路。
> 
> 老来新笋初香，衔得乱莺最苦。
> 
> 蔗浆除热，剩留赠、歌唇樊素。
> 
> 认此心、玛瑙春痕，颗颗泪绡难数。
