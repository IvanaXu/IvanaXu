> 复病三首
>
> 宋·苏辙
>
> 病作日短至，病消秋气初。
> 
> 山深足氛瘴，俗俭少肴蔬。
> 
> 药乱曾何补，心安当自除。
> 
> 朝廷闵流落，已是脱迁居。
> 
> 寒作埋冰雪，热攻投火汤。
> 
> 今生那有此，宿业未应亡。
> 
> 委顺一无损，力争徒自伤。
> 
> 颓然付一榻，是处得清凉。
> 
> 一病五十日，复尔当解官。
> 
> 不才归亦乐，无食去犹难。
> 
> 黾勉人应笑，低徊意已阑。
> 
> 旧师摩诘老，把卷静中看。
