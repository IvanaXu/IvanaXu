> 天梁之什二章
>
> 明·王世贞
>
> 郁彼崇墉，天梁造空。
> 
> 矫如飞龙，脩如螮蝀。
> 
> 越岭汎江，以达大邦。
>

> 
> 焕若霞绡，伟若云寮。
> 
> 自淮徂漕，以达大朝。
> 
> 屯兮屯兮，兆人之人也噫。
