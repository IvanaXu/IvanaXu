> 玩月诗
>
> 唐·曲龙山仙
>
> 曲龙桥顶玩瀛洲，凡骨空陪汗漫流。
> 
> 不假丹梯蹑霄汉，水晶盘冷桂花秋。
> 
> 月砌瑶阶泉滴乳，玉箫催凤和烟舞。
> 
> 青城丈人何处游，玄鹤唳天云一缕。
> 
> 造化天桥碧海东，玉轮还过辗晴虹。
> 
> 霓襟似拂瀛洲顶，颢气潜消橐籥中。
> 
> 危桥横石架云端，跨鹿登临景象宽。
> 
> 颢魄洗烟澄碧落，桂花低拂玉簪寒。
