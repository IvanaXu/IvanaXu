> 句
>
> 宋末金初·王予可
>
> 石裂雯华渍月秋。
> 
> 壶树苔波月渍皴。
> 
> 一壶天地醒眠小。
> 
> 万叠云山飞小雁。
> 
> 风色偃貂裘。
> 
> 唾尖绒舌淡红甜。
> 
> 金盆水不暖，翠雀啄晴苔。
> 
> 凤蹴瑶华散，龙衔桂子香。
> 
> 一片冷截潭底月，六弯斜卷陇头云。
> 
> 帘捲绿阴花外月，玉山冰雪醉扶翁。
> 
> 啼鸟倒衔金羽舞，惊蛇斜傍玉帘飞。
> 
> 凤环捧席带香屏，鲸杯倚伎和云捲。
