> 無對有，實對虛，作賦對觀書。綠窗對朱戶，寶馬對香車。伯樂馬，浩然驢，弋雁對求魚。分金齊鮑叔，奉璧藺相如。擲地金聲孫綽賦，迴文錦字竇滔書。未遇殷宗，胥靡困傅巖之築；既逢周後，太公舍渭水之漁。
