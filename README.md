# བཀྲ་ཤིས་བདེ་ལེགས་
> 春夜宴桃李園序
> 
> 唐代：李白 
> 
> 夫天地者，萬物之逆旅也；光陰者，百代之過客也。而浮生若夢，爲歡幾何？古人秉燭夜遊，良有以也。況陽春召我以煙景，大塊假我以文章。會桃花之芳園，序天倫之樂事。羣季俊秀，皆爲惠連；吾人詠歌，獨慚康樂。幽賞未已，高談轉清。開瓊筵以坐花，飛羽觴而醉月。不有佳詠，何伸雅懷？如詩不成，罰依金谷酒數。(桃花 一作：桃李)
