> 石榴花辞
>
> 元末明初·郯韶
>
> 石榴花，为谁好，一树垂垂向官道。
> 
> 年年花落复花开，不觉前年人易老。
> 
> 去年花开白日长，长官载酒来称觞。
> 
> 流莺蛱蝶共飞舞，不惜醉倒花树傍。
> 
> 今年花开秋可怜，累累结子满树颠。
> 
> 无人载酒花下饮，只说花开胜去年。
> 
> 去年今年两结子，雨露生成有如此。
> 
> 人老不似花再开，莫惜载酒花前来。
