>
> 元末明初·刘基
>
> 江上秋来惟有雨，江城九月犹炎暑。

> 
> 深闭户，莎鸡露泣寒螀语。
>

> 
> 玉帐悠悠闲白羽。
> 
> 愁正聚，乱鸦啼破楼头鼓。
