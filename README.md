> 发归濑三瀑布望两溪诗
>
> 南北朝·谢灵运
>
> 我行乘日垂，放舟候月圆。
> 
> 沫江免风涛，涉清弄漪涟。
> 
> 积石竦两溪，飞泉倒三山。
> 
> 亦既穷登陟，荒蔼横目前。
> 
> 窥岩不睹景，披林岂见天。
> 
> 阳鸟尚倾翰，幽篁未为邅。
> 
> 退寻平常时，安知巢穴难。
> 
> 风雨非攸吝，拥志谁与宣。
> 
> 倘有同枝条，此日即千年。
