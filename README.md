>
> 清·顾贞立
>
> 曾向雪中开。
> 
> 没点尘埃。
> 
> 肯随桃李逐波来。
> 
> 为有湘灵深遣护，掷去重回。
> 
> 无力傍瑶钗。
> 
> 泪粉盈腮。
> 
> 生香真色莫疑猜。
> 
> 好伴杨枝甘露水，供养莲台。
> 
> 绿萼试清泉。
> 
> 香粉增妍。
> 
> 休言海筏度无船。
> 
> 春色原如秋水冷，未许人怜。
> 
> 若过散花天。
> 
> 风韵依然。
> 
> 一枝弥勒现尊前。
> 
> 不羡人间双并蒂，愿化青莲。
