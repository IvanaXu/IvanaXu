> 恭對慢，吝對驕，水遠對山遙。鬆軒對竹檻，雪賦對風謠。乘五馬，貫雙鵰，燭滅對香消。明蟾常徹夜，驟雨不終朝。樓閣天涼風颯颯，關河地隔雨瀟瀟。幾點鷺鷥，日暮常飛紅蓼岸；一雙鸂鶒，春朝頻泛綠楊橋。
