> 晉獻公殺世子申生
> 
> 先秦：佚名 
> 
> 晉獻公將殺其世子申生，公子重耳謂之曰：“子盍言子之志於公乎?”世子曰：“不可。君安驪姬，是我傷公之心也。”曰：“然則盍行乎?”世子曰：“不可。君謂我欲弒君也。天下豈有無父之國哉?吾何行如之?”
> 
> 使人辭於狐突曰：“申生有罪，不念伯氏之言也，以至於死。申生不敢愛其死。雖然，吾君老矣，子少，國家多難。伯氏不出而圖吾君，伯氏苟出而圖吾君，申生受賜而死。”再拜稽首，乃卒。是以爲恭世子也。
