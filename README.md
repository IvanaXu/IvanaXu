> 和左丞庾杲之移病诗
>
> 南北朝·沈约
>
> 岁暮岂云聊，参差忧与疾。
>

>

> 
> 安用谈天辩，徒劳梦赐笔。
> 
> 挂冠若东都，山林宁复出。
