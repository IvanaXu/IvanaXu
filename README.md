> 瑶华 感事作
>
> 当代·问云
>
> 东风锦幕，彩仗轮辉，报新翻传帖。
> 
> 披红石兽，长荫里、顿染唐花香馥。
> 
> 熏弦拨断，做愁听、谁哀空劫。
> 
> 待扫尘、深巷迷烟，冷落余霞霜发。
> 
> 煌煌火树连星，斗朱槛瑶宫，遍试衢陌。
> 
> 萤灯忆雪，开夜户、不奈寒虫鸣壁。
> 
> 总归暮槿，瞬眼过、虚华重歇。
> 
> 更几番、饮罢摛毫，坐起书生肠绝。
