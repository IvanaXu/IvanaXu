> 長相思·其一
> 
> 李白
> 
> 長相思，在長安。
> 
> 絡緯秋啼金井闌，微霜悽悽簟色寒。
> 
> 孤燈不明思欲絕，卷帷望月空長嘆。
> 
> 美人如花隔雲端！
> 
> 上有青冥之長天，下有淥水之波瀾。
> 
> 天長路遠魂飛苦，夢魂不到關山難。
> 
> 長相思，摧心肝！
>
