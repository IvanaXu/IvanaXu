> 信陵馆酒间二首 其一
>
> 金·雷琯
>
> 閒过信陵饮，有怀信陵君。
> 
> 君去日已远，谁怜抱关人。
> 
> 径携一壶酒，往酹公子坟。
> 
> 坟科久已平，墓木几为薪。
> 
> 泉扉锁长夜，千载不复晨。
> 
> 昔与贤俊游，今为狐兔邻。
> 
> 豪贵竟安在，念之心纷纭。
> 
> 有生会归尽，但恐后无闻。
> 
> 此意不可必，且醉梁园春。
