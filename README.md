# བཀྲ་ཤིས་བདེ་ལེགས་
> 唐雎說信陵君
> 
> 先秦：佚名 
> 
> 信陵君殺晉鄙，救邯鄲，破秦人，存趙國，趙王自郊迎。
> 
> 唐雎謂信陵君曰：“臣聞之曰：事有不可知者，有不可不知者；有不可忘者，有不可不忘者。”信陵君曰：“何謂也？”對曰：“人之憎我也，不可不知也；吾憎人也，不可得而知也。人之有德於我也，不可忘也；吾有德於人也，不可不忘也。今君殺晉鄙，救邯鄲，破秦人，存趙國，此大德也。今趙王自郊迎，卒然見趙王，願君之忘之也。”信陵君曰：“無忌謹受教。”
