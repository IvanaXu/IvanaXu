> 叔侄
> 
> 曰諸父，曰亞父，皆叔父之輩；曰猶子，曰比兒，俱侄兒之稱。阿大中郎，道韞雅稱叔父；吾家龍文，楊素比美侄兒。烏衣諸郎君，江東稱王謝之子弟；吾家千里駒，符堅羨苻朗爲侄兒。竹林叔侄之稱，蘭玉子侄之譽。
> 
> 存侄棄兒，悲伯道之無後；視叔猶父，羨公綽之居官。盧邁無兒，以侄而主身之後；張範遇賊，以子而代侄之生。
>
