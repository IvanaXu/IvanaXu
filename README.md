> 沁园春 其二
>
> 元末明初·邵亨贞
>
> 漆点填眶，凤梢侵鬓，天然俊生。
> 
> 记隔花瞥见，疏星炯炯，倚阑延伫，止水盈盈。
> 
> 端正窥帘，瞢腾凭枕，睥睨檀郎长是青。
> 
> 销凝久，待嫣然一顾，密意将成。
> 
> 困酣时倚银屏。
> 
> 强临镜、匆匆犹未醒。
> 
> 忆帐中亲睹，似嫌罗密，尊前斜注，翻怕镫明。
> 
> 醉后看承，歌时斗弄，几度孜孜频送情。
> 
> 难忘处，是香罗揾透，别泪双零。
