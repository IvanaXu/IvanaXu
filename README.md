> 諫院題名記
> 
> 宋代：司馬光 
> 
> 古者諫無官，自公卿大夫，至於工商，無不得諫者。漢興以來，始置官。
> 
> 夫以天下之政，四海之衆，得失利病，萃於一官使言之，其爲任亦重矣。居是官者，常志其大，舍其細；先其急，後其緩；專利國家而不爲身謀。彼汲汲於名者，猶汲汲於利也，其間相去何遠哉！
> 
> 天禧初，真宗詔置諫官六員，責其職事。慶曆中，錢君始書其名於版，光恐久而漫滅。嘉祐八年，刻於石。後之人將歷指其名而議之曰：“某也忠，某也詐，某也直，某也曲。”嗚呼！可不懼哉！
