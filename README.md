> 酬学诗僧惟晤
>
> 宋·欧阳修
>
> 诗三百五篇，作者非一人。
> 
> 羁臣与弃妾，桑濮乃淫奔。
> 
> 其言苟可取，庞杂不全纯。
> 
> 子虽为佛徒，未易废其言。
> 
> 其言在合理，但惧学不臻。
> 
> 子佛与吾儒，异辙难同轮。
> 
> 子何独吾慕，自忘夷其身。
> 
> 苟能知所归，固有路自新。
> 
> 诱进或可至，拒之诚不仁。
> 
> 维诗于文章，太山一浮尘。
> 
> 又如古衣裳，组织烂成文。
> 
> 拾其裁剪馀，未识衮服尊。
> 
> 嗟子学虽劳，徒自苦骸筋。
> 
> 勤勤袖卷轴，一岁三及门。
> 
> 惟求一言荣，归以耀其伦。
> 
> 与夫荣其虑，不若启其源。
> 
> 韩子亦尝谓，收敛加冠巾。
